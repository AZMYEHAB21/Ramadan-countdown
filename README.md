# 🌙 عداد رمضان 2027

موقع بسيط وأنيق لعرض العد التنازلي لبداية شهر رمضان 2027، تم تطويره باستخدام HTML و CSS و JavaScript، مع تصميم جذاب وتأثيرات بصرية جميلة.

---

## 📌 عن المشروع

هذا المشروع تم تنفيذه بشكل فردي كجزء من التدريب العملي على تطوير واجهات المستخدم (Frontend)، بهدف تحسين المهارات في:

* JavaScript (التعامل مع الوقت والتواريخ)
* تصميم واجهات المستخدم (UI)
* إنشاء مواقع متجاوبة (Responsive Design)

## 🚀 طريقة التشغيل

1. قم بتحميل المشروع أو عمل Clone:

```bash

```

2. افتح المجلد الخاص بالمشروع

3. شغل الملف:

```bash
index.html
```

## ⚙️ طريقة العمل

* يعتمد العداد على تاريخ بداية رمضان:

```js
February 8, 2027
```

* يتم حساب الفرق بين:

  * الوقت الحالي
  * وقت بداية رمضان

* يتم تحديث العداد كل ثانية باستخدام:

```js
setInterval(updateCountdown, 1000);
```

* عند وصول العداد للصفر:

  * يتوقف العداد
  * تظهر رسالة تهنئة
  * تبدأ تأثيرات الاحتفال 🎉
